REPLACE THIS FILE WITH YOUR LOGO

The Brand component (components/shell/Brand.tsx) reads its raster
master from `/brand/bb-logo.png` (this folder, this filename).

To use your custom monogram:
  1. Save your logo as a transparent-background PNG
     (recommended 1024x1024)
  2. Drop it here as `bb-logo.png` (overwrites this README)
  3. Hard-refresh the browser (Ctrl+F5)

If the file is missing or fails to load, the LogoMark component
silently falls back to a hand-built SVG double-B in gold.

The wordmark "BUMPERBID", the gold "Auctions" subtitle, and the
"A unit of Zidan Auto Pvt Ltd." tagline are rendered in code, not
as part of the image. The PNG should contain the monogram only -
no wordmark, no tagline, no background.
